API de Imagem Astronômica Diária!

Ao clicar no botão Buscar Imagem você será instantaneamente presenteado com uma imagem ou fotografia de nosso magnífico universo. Para uma qualidade superior, basta acionar o botão Buscar imagem HD!. Na caixa "Data:", é possível inserir qualquer data e a imagem correspondente da data especificada será exibida.
ATENÇÃO: A data deve seguir o formato AAAA-MM-DD (Exemplo: 2023-09-16).
